# Fiber Feed 📄🚀

### A MediaStack News **Aggregator**

Developed with Go and the Fiber framework. It fetches and displays the latest news articles from various sources, providing users with an up-to-date and comprehensive news experience.

## Key Features ✨

- **Real-time News Aggregation 📰**: Continuously fetches the latest news articles from multiple sources to keep users informed.

- **High Performance ⚡**: Utilizes the Fiber framework to ensure fast and efficient handling of HTTP requests.

- **Responsive Design 📱**: Ensures a seamless experience across devices with a mobile-friendly interface.

- **Dockerized Deployment 🐳**: Simplifies deployment using Docker, allowing for consistent environments across different systems.

## Technologies Used 🛠️

- **Go**: The core programming language used for developing the application.

- **Fiber**: A Go web framework inspired by Express, providing a robust set of features for web development.

- **Docker**: Containerization platform used to package the application for deployment.

- **MediaStack API**: External service used for fetching news data.
